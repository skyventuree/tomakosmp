Formerly known as MCWithFams, is a private Minecraft Survival Multiplayer for friends to play and have fun. Founded by Rosetta and his friends.

Tomako is the new SMP with new and improved performance Paper server software with Bedrock client support using Geyser and more features coming soon to the server, focusing on improving player experience.

Check out the stuff below on how to join the server.

Server IP

Please use Java 1.17.1 or Bedrock 1.17.X. Other versions might not be compatible with the server software, hence preventing you from connecting you to the server.

tomakosmp.ddns.net

Keep the port as it is, do not change them.

Scroll down if you don't know how to add server IP to your server list.
